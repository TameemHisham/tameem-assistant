# 🧠 Tameem Assistant

## 🚀 Project Overview

**Tameem Assistant** is a personal full-stack AI-powered assistant designed to help users manage tasks, learn topics, and automate workflows.

It integrates:

- ⚡ **FastAPI** backend for logic and APIs
- 💻 **React (Next.js)** frontend for user interaction
- 🗄️ **PostgreSQL** database for persistence
- 🤖 **AI modules** for personalization and intelligent responses

---

## software versions

Version Python: Python 3.13.7
Version Node: v24.4.0
Version Docker: Docker version 28.3.0

## ⚙️ Run Commands

### 🐍 Backend

```bash
# Activate virtual environment
source venv/bin/activate

# Run FastAPI app
uvicorn app.main:app --reload
```

### 💻 Frontend

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## 🧰 Tech Stack

### 🖥️ Backend

- **FastAPI** — high-performance API framework
- **SQLAlchemy** — ORM for database models
- **PostgreSQL** — relational database
- **Pydantic** — data validation
- **JWT** — authentication

### 🌐 Frontend

- **React** — UI framework
- **Tailwind CSS** — styling
